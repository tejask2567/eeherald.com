HiSilicon extends technology partnership with Vivante on graphics IP

Vivante Corporation has announced that HiSilicon has expanded its ongoing technology partnership to include additional GC (Graphics and Compute) cores. HiSilicon will continue to integrate these key pieces of visual computing IP into SoC products for mobile and consumer products.

Access to the latest Vivante IP cores, give HiSilicon an innovative technology platform based on the latest 3D, CGPU (Composition GPU) and GPGPU APIs. The latest agreement enables HiSilicon to deliver the highest graphics performance in products spanning its entire portfolio.

Vivante GPUs feature the ScalarMorphic unified shader architecture that increases performance for multi-threaded applications that can dynamically balance workloads. GC cores uses cutting edge GPU architecture, advanced unified graphics shaders, and low power methodologies. GC cores support the latest graphics and compute APIs from Khronos, Microsoft and Google.
