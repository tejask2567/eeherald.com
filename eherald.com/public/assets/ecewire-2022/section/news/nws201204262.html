Wi-Fi design code by GainSpan for TI MSP430

GainSpan Corporation has introduced reference design code to include Wi-Fi connectivity to embedded systems which is based upon Texas Instruments' MSP430 microcontroller (MCU).

Using AT commands, the designers can communicate with a GainSpan Wi-Fi module through a serial interface. The MSP430 based products can be added with GainSpan Wi-Fi connectivity and the networking features like enterprise security, embedded DHCP, DNS and HTTP (S) servers, wireless provisioning, and over the air firmware upgrades.

The reference design code running on the MSP430 includes both SPI and UART drivers, a hardware adaptation layer (HAL), and an MCU-independent application layer that includes the AT library. The application code includes UDP, TCP and HTTP traffic, supports multiple concurrent sockets and AP functionality. This code is portable to other host microcontrollers.

The MSP430 reference code uses less than 8 Kbytes of flash and 2 Kbytes of RAM when including the drivers, UDP, TCP and multiple sockets. It can be used with the IAR embedded workbench kickstart edition.

The MSP430 reference design code can be downloaded from GainSpan's website http://www.gainspan.com
